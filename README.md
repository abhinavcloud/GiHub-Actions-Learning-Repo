# Practice Project 2

In this project we are practicing Actions Types and Event Filters of GitHub Actions workflows
We will build 4 workflows. 

### Workflow 1 (workflow1.yml)
Workflow with an Action Type on the Event

### Workflow 2 (workflow2.yml)
Workflow with an Event Filter on the Event

### Workflow 3 (workflow3.yml)
Workflow with an Event filter and Action Type on the Event (both conditions should be satisified)

### Workflow 4 (workflow4.yml)
Workflow with and Event filter and Action type on the Event (any of the conditions sould be satisfied)

#### Note: Documentation of each workflows will be part of the workflow yaml with proper comments.
